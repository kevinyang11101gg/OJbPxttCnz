# Python校园学生宿舍管理系统

## 核心功能
- 登录注册
- 寝室信息管理
- 宿舍卫生管理
- 水电费管理
- 晚归登记管理
- 公告信息
- 学生管理
- 个人中心
- 系统管理

## 适用场景
- 校园宿舍管理
- 学生宿舍信息维护
- 宿舍卫生及水电费管理
- 学生住宿信息查询与维护
- 晚归记录及处理

## 技术特点
- 使用 Django 框架进行后端开发
- 基于 MySQL 数据库进行数据存储
- 前端采用 Vue.js 技术
- 源码附带详细文档及演示 PPT
